# ReminderApi — сервис-напоминалка

## Назначение веб-сервиса
ReminderApi — это веб-сервис для создания, хранения и управления напоминаниями о событиях и задачах. Сервис предоставляет программный интерфейс (Web API), позволяющий клиентским приложениям работать с напоминаниями через HTTP-запросы.

---

## Краткое описание решаемой задачи и целевой аудитории
Веб-сервис решает задачу централизованного хранения и обработки напоминаний, обеспечивая возможность создания новых напоминаний и получения списка уже существующих.

Сервис может использоваться как самостоятельное решение либо как часть более крупной информационной системы (например, системы планирования задач или личного органайзера).

**Целевая аудитория сервиса:**
- студенты;
- сотрудники организаций;
- пользователи, которым важно планировать своё время и контролировать выполнение задач;
- разработчики, использующие API для интеграции функциональности напоминаний в свои приложения.

---

## Основные функции веб-сервиса
Веб-сервис ReminderApi реализует следующие основные функции:

- **Создание напоминаний**  
  Позволяет создавать новые напоминания с указанием названия, описания, даты и времени выполнения, приоритета и статуса.

- **Получение списка напоминаний**  
  Обеспечивает получение списка всех созданных напоминаний в формате JSON.

- **Взаимодействие через Web API**  
  Работа с сервисом осуществляется с использованием HTTP-запросов (GET, POST), что обеспечивает гибкость интеграции с различными клиентскими приложениями.

---

## Архитектурные особенности веб-сервиса
При разработке веб-сервиса использованы следующие технологии и архитектурные решения:

- язык программирования: **C#**;
- платформа: **ASP.NET Core (.NET 8)**;
- архитектурный стиль: **REST**;
- формат обмена данными: **JSON**;
- документирование API: **Swagger (OpenAPI)**;
- хранение данных: временное хранение в оперативной памяти без использования базы данных.

Для документирования и тестирования веб-API используется Swagger UI, который автоматически формирует описание доступных эндпоинтов и моделей данных.

В рамках дальнейшего развития сервиса предполагается возможность интеграции со сторонними API, в том числе с сервисом генерации отчётов. Такой сервис может использоваться для формирования сводной аналитической информации по напоминаниям пользователя за выбранный период (например, количество выполненных и предстоящих задач).

## Инструкция по развертыванию и запуску веб-сервиса
Для запуска веб-сервиса необходимо выполнить следующие шаги:

1. Открыть проект **ReminderApi** в среде разработки **Visual Studio**.
2. Убедиться, что установлен **.NET SDK версии 8.0**.
3. Запустить проект в режиме **Debug**.
4. После запуска открыть браузер и перейти по адресу:
http://localhost:8080/swagger
5. Использовать интерфейс Swagger UI для просмотра документации и тестирования эндпоинтов веб-API.
